# AI MEMORY SYSTEM (docs/ai/)

## ¿Para qué sirve esta carpeta?
Este directorio actúa como la **Memoria Persistente Multi-Agente** del proyecto Oftalmología Si2.
Es el puente cognitivo integral para administrar y desarrollar un sistema robusto distribuido en Web, Backend y **típicamente Aplicaciones Móviles (Flutter / Mobile)** simultáneamente.

## Instrucciones y Orden de Lectura para Agentes de Desarrollo:
1. **`MASTER_AGENT_PROMPT.md`** - Lee las reglas inquebrantables del ecosistema multiplataforma.
2. **`PROJECT_VISION.md`** - Internaliza y acopla tu lógica a qué sistema y problema estamos abordando.
3. **`ARCHITECTURE.md` y `MOBILE_ARCHITECTURE.md`** - Domina las directrices macro y micro de cómo se interconectan contenedores, apis y librerías entre la PC, la tabla de Base de Datos y los Celulares de destino.
4. **`TECH_STACK.md`** - El arsenal tecnológico en sus versiones aprobadas.
5. **`CURRENT_STATE.md`** y **`HANDOFF_LATEST.md`** - Estado vigente y última sesión (API mobile, login por email, citas paciente, Docker/networking).
6. **`NEXT_STEPS.md`** - Escoge o suma metas.
7. **`DECISIONS_LOG.md`** - Decisiones técnicas recientes (JWT, ALLOWED_HOSTS, Dio base URL, filtrado de citas).

**Jamás cierres o indiques al operador humano "TAREA FINALIZADA" sin habernos hecho el traspaso a los otros agentes, actualizando esta carpeta correspondientemente.**
